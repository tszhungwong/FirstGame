# Game Ghost 正上方俯視地圖 Sample 設計

## 目標

製作兩張可作為遊戲關卡美術方向參考的完整地圖 sample。兩張地圖皆採真正 90° 正上方俯視，延續 `style.md` 的精緻 Q 版東方幻想語言，並確保所有玩法地形在縮小後仍容易辨識。

## 共通視覺規格

- 高完成度 2D Q 版遊戲概念插畫。
- 乾淨細緻、略帶深棕或深灰的墨線；外輪廓稍重，內部細節較輕。
- 柔和水彩材質結合乾淨賽璐璐陰影，明暗清楚但不寫實。
- 圓潤集中、易讀的地形輪廓；細節中心豐富、邊緣收斂。
- 東方幻想弧形與雲紋細節，配合少量輕科幻機械接縫和青色發光焦點。
- 完整單張競技場構圖，尺寸與視覺密度相近。
- 不放角色、UI、文字、標籤、Logo 或水印。
- 不採等角、斜俯或三分之四視角。

## 地圖一：草地／森林

- 主色為自然草綠、森林深綠、河流藍與泥土棕，控制大面積高飽和色。
- 深綠、蓬鬆且輪廓清楚的草叢形成數個可藏身區。
- 藍色河流完整切斷部分行動路線，河岸輪廓清楚，不與普通地面混淆。
- 棕褐色泥潭以黏稠表面、足跡或緩慢漣漪表達減速功能。
- 大型樹冠與粗樹幹形成射線遮擋；樹木位置不能完全遮住主要通路。
- 地形組合形成可繞行、可伏擊且有清楚戰術選擇的競技場。

## 地圖二：廢棄倉庫

- 主色為鏽棕、槍鐵灰、舊木色與少量青藍發光焦點。
- 使用磨損金屬地板、鏽蝕接縫和舊警示紋表達廢棄工業空間。
- 淡青灰色迷霧形成數個藏身區；霧區邊界柔和，但範圍必須清楚。
- 木箱、金屬箱和貨櫃形成射線遮擋，輪廓與地面有明確分離。
- 輸送帶使用清楚的方向箭頭、規律分段與青色流動光帶表達加速方向。
- 箱子、迷霧與輸送帶形成可追逐、伏擊和快速轉移的路線。

## 交付與驗收

- 輸出兩張獨立 PNG 圖片並保存於專案資產目錄。
- 圖一同時清楚包含草叢、河流、泥潭和樹木。
- 圖二同時清楚包含迷霧、箱子和輸送帶。
- 在不依賴文字說明的情況下，各類地形功能仍能由造型、材質和顏色區分。
- 兩張圖的視角、完成度、線條與上色風格一致。
