# Source concept art

These images are production references only. They are never imported into the Godot project or used as runtime sprites. Approved derivatives must be created under `game/assets/runtime/` and registered separately in `docs/assets/asset-register.csv`.

The character and item sheets are AI-assisted concept studies based on the prompts recorded in `docs/assets/ai-prompts.md`. The map sheets are AI-assisted layout studies. All nine source images remain tracked by Git LFS.
